  1. Documentação Python 3 (gratuito, português, documentação)

    A documentação oficial do Python 3 está repleta de explicações e detalhes técnicos sobre a linguagem de programação. Muito útil para quem está aprendendo ou já é um profissional experiente em Python e busca estar sempre atualizado.

  2. PyPI (gratuito, inglês, repositório de código)

    Python Package Index (PyPI) é o repositório oficial de pacotes de software para a linguagem de programação Python. Os usuários podem buscar por diferentes pacotes e inclusive disponibilizar seus próprios.

  3. Lista de Exercícios Python Brasil (gratuito, português, código)

    Uma extensa lista de exercícios de Python, dividida em categorias e subcategorias. Ideal para quem está aprendendo a programar em Python e quer colocar a mão na massa.

  4. List Comprehension em Python (gratuito, inglês, texto)

    Neste artigo, o autor explica como o uso de list comprehension pode tornar o código Python mais limpo e eficiente. Ideal para os programadores que querem aperfeiçoar seus conhecimentos na linguagem.

  5. Função zip em Python (gratuito, inglês, texto)

    Este é um tutorial sobre a função integrada zip do Python, que é usada para combinar iteráveis. O tutorial aborda diferentes aspectos da função e apresenta exemplos de seu uso.

  6. zip em Python (gratuito, inglês, texto)

    Um guia explicativo sobre a função zip no Python. O conteúdo detalha como a função funciona, para que serve e como aplicá-la corretamente.

  7. Exceções em Python (gratuito, inglês, texto)

    Um guia detalhado sobre como lidar com exceções em Python. O conteúdo é bem explicativo e oferece exemplos práticos para ajudar na compreensão do leitor.
